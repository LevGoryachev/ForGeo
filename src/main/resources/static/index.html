ForGeo

REST service for addresses and geolocations of company facilities

CRUD for addresses and/or location points (address-location can be used both separately and together)

API v3

  • GET /api/v3/addresses
  • GET /api/v3/addresses?type=typeName
  • GET /api/v3/addresses/id
  • POST/api/v3/addresses
  • PUT /api/v3/addresses
  • DELETE /api/v3/addresses/id
  • GET /api/v3/locations
  • GET /api/v3/locations/id
  • POST /api/v3/locations
  • PUT /api/v3/locations
  • DELETE /api/v3/locations/id
